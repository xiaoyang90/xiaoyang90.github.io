自定义函数

  1. 定义函数
    在用python编写程序过程中，有时会根据需要自己定义函数。在python中，定义一个函数要使用def语句，一次写出函数名，小括号，参数，以及冒号。在缩进块中编写函数体，函数的返回值用return语句返回。
    例如我们自定义一个求绝对值的函数:

    def my_abs(x):

    if x>= 0:
    return x
else:
    return -x

    这里打印看一下-10的绝对值

    print my_abs(-10)
    结果输出为10
    当程序在函数体内执行时，当执行到return语句时，函数执行完毕，并将结果返回。但如果没有return语句，函数执行完毕后也会返回。但返回的结果是None。
    如果已经将定义的函数保存为.py文件了，例如，将my_abs保存为abstest.py文件了，则可以在该文件的当前目录下，启用Python解释器，用from abstest import my_abs来导入此函数。
    函数可以同时返回多个值，但其实时返回的一个元组tuple。
