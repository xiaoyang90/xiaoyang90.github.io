Python 中文显示

在使用Pycharm编写Python程序时，在程序中出现中文，但却无法显示中文。在settings中将IDE Encoding改为UTF-8编码，并且在程序开头使用：

# -*- coding:utf-8 -*-

声明编码方式为utf-8编码，并且在中文之前添加u，例如：

plt.xlabel(u"时间")

但是仍然出现无法正常显示中文的情况。
这时，从matplotlib.font_manager模块中导入FontProperties:

from matplotlib.font_manager import FontProperties

并且通过以下语句添加中文字体，并且可以更改字体大小：

font = FontProperties(fname = r"C:\windows\fonts\simsun.ttc",size = 20)

此时：

plt.xlabel(u"时间",fontproperties=font)

就可以正常显示中文了。
