1、高阶函数 Higher-order function
函数的参数可以接受变量，而变量又可以指向函数，那么一个函数就可以接受另一个函数作为参数，这种函数称之为高阶函数。
一个简单的高阶函数：

def multiply(x,y,f):
    return f(x) *f(y)

2.map函数
map函数接受两个参数，第一个参数是函数，另一个是Iterable，map将传入的函数依次作用到序列的每个元素，并把结果作为新的Iterator，Iterator为一个惰性序列，可以通过list（）将整个序列都显示出来并返回一个list。
例如：

def f(x):
    return x * x *x
r = map(f,[1,2,3])
print list(r)

利用map（）函数，把用户输入的不规范的英文名字，变为首字母大写，其他小写的规范 名字。输入：[‘adam’,’LISA’,’barT’], 输出：[‘Adam’,’Lisa’,’Bart’]

3.reduce函数
reduce也是将一个函数作用于一个序列，但是这个函数必须接受两个参数，reduce把结果继续和序列的下一个元素做累积计算，记过就是：

reduce(f,[x1,x2,x3,x4]) = f(f(f(x1,x2),x3),x4)

例如，我们想要将一个序列[1,2,3,4,5]变为12345.实现方法：

from functools import reduce
def f(x,y):
    return 10 * x + y
print reduce(f,[1,2,3,4,5])

定义一个函数，用于求列表中所有数的乘积

def prod(L):
    def f(x,y):
        return  x * y
    return reduce(f,L)
L = [2,3,4,5]
print prod(L)

4.filter函数
filter函数也接受一个函数以及一个序列，但是filter（）把传入的函数一次作用于每个元素，然后根据返回值是True还是Flase决定保留还是丢弃该元素。
filter（）函数返回的时一个Iterator，即惰性序列，所以要强迫filter（）完成计算，需要用到list（）函数 获得所欲结果并返回。
