Python基础学习3

1、什么是lambda?
我们可以认为lambda定义了一个匿名函数。例如：

g = lambda x:x+1
print g(1)

上述代码x为入口参数，x+1为函数体。这样使得函数更加简洁，直观。
2、排序算法
Python内置的sorted（）函数就可以对list进行排序:例如：

sorted（[1,2,3,2,1]）

sorted（）函数也是一个高阶函数，可以接受一个key函数来实现自定义排序。例如，按照绝对值大小进行排序：

Ｌ＝　［－４，２，１，８，－７］
print sorted (L,key = abs)

key指定的函数将作用与list的每一个元素上，并根据key函数返回的结果进行排序。然后sorted（）按照keys（）进行排序，并且按照对应关系返回到list相应的元素。

默认情况下对字符串排序是按照ASCII大小进行的。但如果要对字符串进行排序，应该用key函数将字符串大小映射为忽略大小写。如果要进行饭箱排序，可以掺入第三个参数，reverse= True。例如：

sorted（["Yang","du",""zhang"，"wang"],key = str.lower,reverse = True）

3返回函数
高阶函数除了可以将函数作为参数以外，还可以将函数作为返回值。
例如：下面是一个参数为可变参数的求和函数

def sum(*args):
    s = 0
    for i in args:
        s = s + i
    return s

4、匿名函数
当我们传入函数时，不需要显示的定义函数，直接传入匿名函数更方便，而且使用匿名函数的另一个优点是，不必担心函数名冲突。例如：

list([map(lambda x:x * x,[1,2,3,4])])

lambda表示匿名函数，冒号前表示函数参数，匿名函数也可以赋值给一个变量，然后通过变量啦调用该函数。
5.偏函数
Python的模块functools的功能很多，其中一个就是偏函数，partial function。
functools.partial就是创建一个偏函数，例如，创建一个将二进制转换为整数的函数：

import functools
int2 = functools.partial(int,base = 2)

functools.partial的作用就是，把一个函数的某些参数设为默认参数，返回一个新的函数，调用这个新的函数更加方便。
base = 2相当与固定了int()函数的关键字参数。

max2 = functools.partial(max,10)
max2(5,6,7)

上面程序的运算结果是10。实际上是把10作为*args的一部分自动加到左边。
相当于：

args= （10，5,6,7）
max(*args)

2.map函数
map函数接受两个参数，第一个参数是函数，另一个是Iterable，map将传入的函数依次作用到序列的每个元素，并把结果作为新的Iterator，Iterator为一个惰性序列，可以通过list（）将整个序列都显示出来并返回一个list。
例如：

def f(x):
    return x * x *x
r = map(f,[1,2,3])
print list(r)

利用map（）函数，把用户输入的不规范的英文名字，变为首字母大写，其他小写的规范 名字。输入：[‘adam’,’LISA’,’barT’], 输出：[‘Adam’,’Lisa’,’Bart’]

3.reduce函数
reduce也是将一个函数作用于一个序列，但是这个函数必须接受两个参数，reduce把结果继续和序列的下一个元素做累积计算，记过就是：

reduce(f,[x1,x2,x3,x4]) = f(f(f(x1,x2),x3),x4)

例如，我们想要将一个序列[1,2,3,4,5]变为12345.实现方法：

from functools import reduce
def f(x,y):
    return 10 * x + y
print reduce(f,[1,2,3,4,5])

定义一个函数，用于求列表中所有数的乘积

def prod(L):
    def f(x,y):
        return  x * y
    return reduce(f,L)
L = [2,3,4,5]
print prod(L)

4.filter函数
filter函数也接受一个函数以及一个序列，但是filter（）把传入的函数一次作用于每个元素，然后根据返回值是True还是Flase决定保留还是丢弃该元素。
filter（）函数返回的时一个Iterator，即惰性序列，所以要强迫filter（）完成计算，需要用到list（）函数 获得所欲结果并返回。

1.迭代
在Python中，除了像list这种有效标的数据类型可以迭代，即使很多数据类型没有下标，但是，只要是可迭代对象，无论有无下标，都可以进行迭代。比如：dic，字符串等。
但值得注意的是，默认情况下。dic迭代的是key，但是可以通过d.value获取dict的value，同时也可以通过d.items()同时获取key和value。
我们可以通过下面的方式来判断一个对象是否为可迭代对象

from collections import Iterable
isinstance('abc',Iterable)

同时python中还有内置函数enumerate，可以把list变为变为索引-元素值。

如果列表中的元素可以按照某种算法推算出来，这是我们就可以使用生成器：generator。generator是一种一边循环一遍计算的机制。

如果一个函数中出现 yield关键字，那么这个函数就不再是普通的函数，而是一个生成器

generator的执行流程和函数的不一样。generator，在每次调用next（）函数的时候执行，遇到yield语句韩慧，再次执行的时候 从上次yield的语句出继续执行。
例如：定义一个generator。一次返回2,4,6

def even():
    print "step1"
    yield 2
    print "step2"
    yield  4
    print "step3"
    yield  6
e = even()
for i in e:
    print i

4.迭代器
首先我们应该知道什么是可迭代对象：Iterable。包括集合数据类型：list，tuple，dic，set，str，以及generator。可以通过下面的语句判断是不是可迭代对象

from collections import Iterable
isinstance([],Iterable)
isinstance({},Iterable)
isinstance((x for x in range(5)),Iterable)

可以被next()函数调用并不断返回下一个值得对象称为迭代器：Iterator。
可以通过isinstance函数判断一个对象是否是Iterator对象

from collections import Iterator
isinstance(x for x in range(5),Iterable)

生成器都是Iterable对象，但list，dic，str虽然是Iterable，但并不是Iterator
可以通过下面的语句将list，tuple，dic，str变为Iterator

isinstance(iter([]),Iterator)

Iterator表示的是一个数据流，我们可以将数据流看成是一个有序数列，但我们并不知道序列的长度，只能通过next（）函数实现按需计算下一个数据，所以Iterator的计算时惰性的，只有在需要返回下一个数据是才会计算。

Python 中文显示

在使用Pycharm编写Python程序时，在程序中出现中文，但却无法显示中文。在settings中将IDE Encoding改为UTF-8编码，并且在程序开头使用：

# -*- coding:utf-8 -*-

声明编码方式为utf-8编码，并且在中文之前添加u，例如：

plt.xlabel(u"时间")

但是仍然出现无法正常显示中文的情况。
这时，从matplotlib.font_manager模块中导入FontProperties:

from matplotlib.font_manager import FontProperties

并且通过以下语句添加中文字体，并且可以更改字体大小：

font = FontProperties(fname = r"C:\windows\fonts\simsun.ttc",size = 20)

此时：

plt.xlabel(u"时间",fontproperties=font)

就可以正常显示中文了。

FFT

用FFT能将时域的数字信号转换为频域信号。转换为频域信号之后，可以很方便的分析信号的频率成分，在频域上进行处理，最终还可以将处理的频域信号通过IFFT转换为频域信号，实现许多无法再时域完成的信号处理算法。而且对于空间域上的数字图像，也能通过傅里叶变换为频域类的信号。

numpy.fft.rfft(a,n=None,axis=-1,norm=None)

上面的函数值计算以为离散的快速傅里叶分析，其中：a为输入矩阵，n为沿着转换轴的点数，如果n小于输入矩阵的长度，则输入就被剪裁掉，如果n大于输入矩阵，则输入矩阵就会被0填补，如果没有规定n的值，则输入沿着轴的长度由使用的轴给定。如果没给定axis，则使用最后一个轴。返回为一个复杂的ndarray.如果n是偶数 ，转换轴的长度为（n/2）+1.如果n为奇数，则长度为（n+1）/2.

Image thresholding

什么是图像的阈值分割？what is iamge thresholding?


上图中的数字的灰度值和该数相同，255最亮，5最暗。所有的阈值算法都有一个源图像（src），阈值（thresh）最为输入以及一个输出图像（dst），如果src（x，y）>thresh，则dst为指定为某个值，否则，被指定为另外的某些值。
例如：

if src(x,y) > thresh:

    dst(x,y) = maxValue
else:
    dst(x,y) = 0

  1. binary Thresholding(type = THRESH_BINARY)
    Python 

    import cv2  
src = cv2.imread("threshold.png",cv2.IMREAD_GRAYSCALE)  
thresh = 0  
maxValue = 255  
th,dst = cv2.threshold(src,thresh,maxValue,cv2.THRESH_BINARY)

    这是最常用的也是最简单阈值分割类型

  2. *Inverse Binary Thresholding(type =THRESH_BINARY_INV))
    Thresholding rule

    if src(x,y) > thresh
    dst(x,y) = 0
else
    dst(x,y) = maxValue

    Python

    th,dst = cv2.threshold(src,thresh,maxValue,cv2.THRESH_BINARY_INV)

  3. *Thruncate Thresholding(type = THRESH_TRUNC)
    Thresholding rule

    if src(x,y) = thresh
    dst(x,y) = thresh
else
    dst(x,y) = src(x,y)

    Pyton

    threshold(src,dst,thresh,maxValue,THRES_TRUNC)

  4. *Threshold to Zero(type = THRESH_TOZERO)
    Thresholding rule

    if src(x,y) > thresh
    dst(x,y) = src(x,y)
else
    dst(x,y) = 0

    Python

    threshold(src,dst,thresh,maxvalue,THRESH_TOZERO)

  5. *Inverted Threshold to Zero(type = THRESH_TOZERO_INV)
    Thresholding rule

    if src(x,y) > thresh
    dst(x,y) = 0
else
    dst(x,y) = src(x,y)

    Python

    th,dst = cv2.threshold(src,thresh,maxValue,cv2.THRESH_TOZERO_INV)
