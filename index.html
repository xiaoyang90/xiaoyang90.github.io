Image thresholding

什么是图像的阈值分割？what is iamge thresholding?


上图中的数字的灰度值和该数相同，255最亮，5最暗。所有的阈值算法都有一个源图像（src），阈值（thresh）最为输入以及一个输出图像（dst），如果src（x，y）>thresh，则dst为指定为某个值，否则，被指定为另外的某些值。
例如：
if src(x,y) > thresh:

dst(x,y) = maxValue

else:
dst(x,y) = 0

  1. Binary Thresholding(type = THRESH_BINARY)
    ** Python
    import cv2 src = cv2.imread("threshold.png",cv2.IMREAD_GRAYSCALE) thresh = 0 maxValue = 255 th,dst = cv2.threshold(src,thresh,maxValue,cv2.THRESH_BINARY)
